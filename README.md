# scoop-bucket

Scoop manifests for Obsta Labs tools.

## Install

```powershell
scoop bucket add obstalabs https://github.com/obstalabs/scoop-bucket
scoop install obstalabs/neurorouter
scoop install obstalabs/neurorouter-pro
scoop install obstalabs/workledger
```

## Available Manifests

| Manifest | Description |
|---------|-------------|
| [neurorouter](https://github.com/obstalabs/neurorouter) | Community edition of NeuroRouter — context hygiene for AI coding sessions |
| [neurorouter-pro](https://github.com/obstalabs/neurorouter-pro-dist) | Commercial NeuroRouter Pro binary release |
| [workledger](https://github.com/obstalabs/hiveram-dist) | Hiveram ledger plane for work orders, claims, release bundles, and WEO attribution |

## Notes

- Manifest updates are generated by each product release workflow.
- `neurorouter-pro` binaries are published from the public distribution repository, not from private source archives.
- `workledger` binaries are published from the public Hiveram distribution repository, not from private source archives.
