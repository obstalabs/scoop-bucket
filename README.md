# scoop-bucket

Scoop manifests for Obsta Labs tools.

## Install

```powershell
scoop bucket add obstalabs https://github.com/obstalabs/scoop-bucket
scoop install obstalabs/neurorouter
```

## Available Manifests

| Manifest | Description |
|---------|-------------|
| [neurorouter](https://github.com/obstalabs/neurorouter) | Community edition of NeuroRouter — cleans AI requests before they hit the model |

## Notes

- This bucket currently publishes only public/community manifests.
- Manifest updates are published from the source repository release workflow.
